Após fazer o "clone" do diretório remoto, você terá a versão corrente do repositório.

Para adicioanr arquivos ao repositório:
$ git add arq[ arq2 arq3 ...]

Após adicionar, é necessário fazero commit:
$ git commit -m "Mensagem de atualização"

Atualizar o repositório remoto
$ git push origin master

Para atualizar seu diretório com os arquivos mais recentes do repositório:
$ git pull
